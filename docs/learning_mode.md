# RTSP 客户端学习交流方式

## 学习模式

### 1. 代码设计阶段
- 我通过提问引导你思考
- 你回答问题，我帮你整理设计
- 逐步细化直到设计完成

### 2. 代码实现阶段
- 你按照设计实现代码
- 我审查代码，提出问题和改进点
- 你修改后再次审查
- 直到代码通过审查

### 3. 测试验证阶段
- 创建单元测试
- 运行测试验证功能
- 修复发现的 Bug

### 4. 下一部分
- 进入下一个模块的设计和实现
- 重复上述步骤

---

## 教学特点

| 特点 | 说明 |
|------|------|
| 循序渐进 | 每次只讨论一个概念或问题 |
| 提问引导 | 不直接给答案，通过问题引导思考 |
| 代码审查 | 实现后帮你审查，发现问题及时指出 |
| 实际动手 | 鼓励你亲自编写代码 |
